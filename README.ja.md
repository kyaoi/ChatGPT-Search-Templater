# ChatGPT Search Templater

**ChatGPT Search Templater** は、Webページ上で選択したテキストを、事前に定義したテンプレートに埋め込んで `chatgpt.com` を開くためのChrome拡張機能です。

複数の検索テンプレートを柔軟に管理し、用途に応じて素早くChatGPTでの検索を実行できます。

## 主な機能

- **テンプレートベースの検索**: 選択したテキストを `https://chatgpt.com/?q={TEXT}` のようなテンプレートURLに挿入して開きます。
- **複数テンプレートの管理**:
  - オプションページでテンプレートの追加、編集、削除が可能です。
  - 各テンプレートに、ChatGPTのモデル（`gpt-4o`, `gpt-5`など）、ヒント検索の有無、一時チャットの利用などを設定できます。
- **クエリ入力モード**:
  - コンテキストメニューの「クエリを入力して実行…」から検索前にクエリやモデル、オプションを調整できます。
  - 既定テンプレート向けのショートカット（初期値: `Ctrl+Shift+U` / `Command+Shift+U`）でも同じ入力ダイアログを開けます。
- **簡単なアクセス**:
  - 拡張機能のポップアップからテンプレートを選択して実行できます。
  - 右クリックのコンテキストメニューから検索を実行できます。
  - キーボードショートカット (`Ctrl+Shift+Y` / `Cmd+Shift+Y`) で既定のテンプレートを素早く実行できます。

## 使い方

1. **テキストを選択**: Webページ上で任意のテキストを選択します。
2. **検索を実行**:
   - **ポップアップ**: 拡張機能のアイコンをクリックし、表示されたポップアップから使用したいテンプレートを選択します。
   - **右クリック**: 選択したテキストの上で右クリックし、コンテキストメニューからテンプレートを選択します。
   - **ショートカット**: `Ctrl+Shift+Y`（Macの場合は `Command+Shift+Y`）を押すと、既定のテンプレートで検索が実行されます。
3. **ChatGPTで開く**: 新しいタブで、テンプレートに基づいたURLが `chatgpt.com` で開かれます。

## 設定方法

1. 拡張機能のアイコンを右クリックし、「オプション」を選択します。
2. オプションページが開き、以下の設定が可能です。
   - **テンプレートの管理**: 新しいテンプレートの作成、既存テンプレートの編集・削除。
   - **既定テンプレートの設定**: ショートカットなどで使用されるデフォルトのテンプレートを選択します。
   - **コンテキストメニューの編集**: 右クリックメニューに表示する項目を管理します。

## ビルド方法

開発用にビルドする手順は以下の通りです。

1. **リポジトリをクローン**:
   ```bash
   git clone https://github.com/your-username/chatgpt-search-templater.git
   cd chatgpt-search-templater
   ```

2. **依存関係をインストール**:
   ```bash
   pnpm install
   ```

3. **ビルド**:
   ```bash
   pnpm build
   ```
   `dist` ディレクトリにビルドされたファイルが生成されます。

4. **Chromeに読み込む**:
   - Chromeで `chrome://extensions` を開きます。
   - 「デベロッパーモード」を有効にします。
   - 「パッケージ化されていない拡張機能を読み込む」をクリックし、`dist` ディレクトリを選択します。
